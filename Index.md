## 🚀 How to Deploy

### Option 1 — Open Locally
Simply open `index.html` in any modern browser. No server required.

### Option 2 — GitHub Pages (Free Hosting)
1. Create a GitHub repository (e.g. `jyotsna-portfolio`)
2. Upload `index.html` to the repository
3. Go to **Settings → Pages → Source → main branch / root**
4. Your site will be live at `https://yourusername.github.io/jyotsna-portfolio/`

### Option 3 — Netlify (Free, Instant)
1. Go to [netlify.com](https://www.netlify.com)
2. Drag and drop the `index.html` file
3. Get an instant live URL

### Option 4 — Vercel
1. Go to [vercel.com](https://www.vercel.com)
2. Import your GitHub repo or drag-and-drop
3. Deploy instantly
